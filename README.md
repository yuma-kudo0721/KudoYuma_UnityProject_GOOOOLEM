# GOOOOLEM

## 概要
本作は、パズルでモンスターを召喚して戦う対戦型パズルゲームです。  
さまざまな種類のモンスターが登場し、それぞれ異なる能力などを持っています。
シングルプレイは倒すとどんどんレベルが上がっていく勇者パーティーをしのぎながら
何レベルまで行けるか競うゲームです。自分の城のHPが０になったらゲームオーバーとなります。
マルチプレイは対人戦となります。二人プレイで互いにモンスターを召喚しあって戦います。
相手の城のHPを０にした方の勝利となります。
東京ゲームショウでの展示を想定して制作を開始しました。
キャラクター間の強さに偏りが出ないよう、ステースタや挙動を調整し、
テストプレイを通してバランスを検証・改善しました。
東京ゲームショウでは色々な人たちに実際にプレイしてもらい、楽しんでもらうことができました。

## 遊び方
1. タイトル画面でモード選択
2. ピースとソウルを一緒に消してモンスター召喚
3. 大砲ゲージを貯めて大砲を発射
   
## 上手にプレイする方法
1. モンスターの能力、性質を知る 
2. タイミングよく大砲を発射  
3. モンスターの召喚タイミングを見極める

## プログラム面（こだわり）
- ドラゴンデストロイヤー 
  - モンスター共通クラス・人間共通クラスを継承した職業専用キャラクターとして実装
  - 攻撃処理にCoroutineを使用し、攻撃モーションを段階的に制御 
  - スプライト切り替えとダメージ判定を分離することで、見た目と処理が一致する自然な攻撃表現を実現  
  - 特定の敵に対して強みを持たせることで、「職業らしさ」をプログラム上で明確に表現

- 大砲  
  - 弾の生成・物理挙動だけでなく、発射時の反動演出まで含めて実装
  - スコアを昇順で取得し、ランキング表示
  - 発射時に砲身を後方へずらし、Lerp を用いて元の位置へ滑らかに復帰させることで「撃った感覚」が伝わる反動演出を実装
  - 単なる攻撃処理に留まらず、操作時の手応えや視覚的なフィードバックを重視して制作

- メタルスライム  
  - スライムはソウル1つで召喚される低コストモンスターとして設計
  - 複数ソウルを含めてピースを消した場合のみ亜種が生成される仕様のため、スライムは亜種を生成できないという制約があった
  - 上記制約への代替案として、一定確率でメタルスライムが生成される仕組みを実装
  - 低コスト召喚でありながら、レア出現による期待感と戦略性を付加
  - メタルスライムは通常モンスターとは異なる専用のダメージ処理を持ち、出現時に戦況が変化しやすい設計
  - ゲームルールの整合性を保ちつつ、プレイヤー体験の幅を広げることを意識した実装
